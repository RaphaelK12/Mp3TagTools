Copy Tags

Before Copying tags select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To Copy Tags from ID3v1.1 to ID3v2.x:

Use the shortcut F6 or Copy ID3v1.1 ----> ID3v2.x from the Tools main program menu.

To Copy Tags from ID3v2.x to ID3v1.1:

Use the shortcut F7 or Copy ID3v2.x ----> ID3v1.1 from the Tools main program menu.

Copy ID3v1.1 ----> ID3v2.x

Use the copy tag preferences dialog to select which all tag fields will be copied and click OK, this will copy all or selected ID3v1.1 tags to their corresponding ID3v2.x tags.

Copy ID3v2.x ----> ID3v1.1

Use the copy tag preferences dialog to select which all tag fields will be copied and click OK, this will copy all or selected ID3v2.x tags to their corresponding ID3v1.1 tags.


Note:

When copying tags from ID3v2.x to ID3v1.1 some of the tag information maybe truncated. This is because ID3v1.1 specifications allow only limited amount of information in them. For example Artist tag can contain a maximum of 30 characters, to see the ID3v1.1 specifications visit the ID3 website http://www.id3.org/ .
