Clean Filenames

Before Cleaning filenames select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To clean the filenames and correct their case:

Use the shortcut Ctrl+N or Clean Filenames button or select Clean Filenames from the Tools main program menu.

This will perform the following actions:

  • Removes all %20 and underscores from filenames
  • Remove excess whitespace characters in the filename
  • Capitalize the first letter of all words in the filename
  • Capitalize the next letter following a user defined list of symbols like ()[]{}0123456789.~`!@#$%^&*-_+=|\:;"<>,?/
  • Capitalize the next letter following a user defined list of Prefix words like Mc,O',N',3D
  • Make a user defined list of words upper case DJ,MC,ABBA,MLTR,TLC,LL,II,UB40
  • Make a user defined list of words lower case and,aka,vs,feat

These actions can be customized using the Correct Case page in the Program Preferences dialog.

For example, a mp3 named "(ewan_mcgregor_and)%20nicole_kidman%20_-_%20cOmE_wHat_mAY.MP3" using Clean Filenames will be renamed to "(Ewan McGregor and) Nicole Kidman - Come What May.mp3"
