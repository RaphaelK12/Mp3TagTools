Export Tag Data

Before Exporting tag data select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To export tag data shown in the list columns:

Select Export Data... from the Tools main program menu.

Use the Export tag preferences dialog to select which all tag fields will be exported and click OK, this will export all or selected ID3v1.1 or ID3v2.x tags to a tab delimited text file.

Tab delimited text files are similar to Comma separated value (CSV) files and can be easily imported into most spreadsheet programs like OpenOffice Spreadsheet or MS Excel. This will enable you to make a print out of the list of files and its data in your mp3 collection.
