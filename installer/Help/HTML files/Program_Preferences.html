Program Preferences

To set Program Preferences:

Select Program Preferences from the Options main program menu or from right click menu in the ID3v1.1 or ID3v2.x tag list views.

You can set preferences to:

  • Select ID3v1.1 or ID3v2.x list columns to show or hide.
  • Select ID3v1.1 or ID3v2.x list columns to Export tag data.
  • Set Correct Case options.
  • Set Colors displayed in the ID3v1.1 or ID3v2.x lists.
  • Select ID3v1.1 or ID3v2.x tags to be copied when using Copy Tags.
  • Preserve  date time attribute (Last modified time stamp) for all tag operations. (useful for those who make backups based on that attribute)
  • Automatically load the last used directory on startup of the program.
