Remove Tags

Before Removing tags select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To remove tags from a mp3:

Use the shortcut Ctrl+Delete or Remove Tags button or select Remove Tags from the Edit main program menu.

Depending on the currently selected Tag info Tab Remove Tags will:

 

ID3v1.1 Tab

This will completely remove all ID3v1.1 tags (which according to the ID3v1.1 specification comes to 128 bytes at the end of the mp3 file), thus the file size decreases by 128 bytes.

ID3v2.x Tab

This will completely remove all ID3v2.x tags including any lyrics and pictures. Since the size of ID3v2.x tag varies, the filesizes are decreased by different amounts. Since ID3v2.x tags are stored in the begining of the mp3, after removing all v2.x tags, the MPEG Header column in the mp3 list should now show 0 bytes.
 

Lyrics Tab

Only the lyrics tag of the mp3 will be removed.

Picture Tab

Each mp3 can have more than one picture, of a particular specified Picture Type and Picture Description. The default Picture Type used is "Other". Remove Tags will remove all the pictures the mp3 contains and not just currently displayed picture.

To remove the picture associated with only one Picture Type, use the Remove Picture Button on the Picture tab page.
