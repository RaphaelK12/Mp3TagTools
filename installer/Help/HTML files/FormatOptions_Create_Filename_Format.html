Create Filename Format

Any combination of Filename formats can be created by 2 methods, for example to create the format <Artist> - <Title> :

Using Shortcuts

In the Create/Edit Filename Format box, type in using the numpad key "*1 - *2"  (without the quotes) and the program will automatically replace the shortcuts with "<Artist> - <Title>". You can also paste in the formats.

Any of the following shortcuts can be used in the Create/Edit Filename Format box:

Using Checkboxes

By checking the corresponding <Artist> checkbox, then click  Add Separator button, then check the <Title> checkbox below the Create/Edit Filename Format box.


Note:
  • Do not add the extension of a file ".mp3" while creating a filename format, it is automatically added by the program.
  • You can use the <Ignore> format tag any number of times depending on the way your mp3s are name. For example using the filename format: "<Artist> - <Ignore> - <Title> - <Ignore>" on an mp3 named "ABBA - Greatest Hits - Dancing Queen - 1" will ignore the album name "Greatest Hits" and track number "1" but will get "ABBA" as <Artist> and "Dancing Queen" as the <Title>.
  • It  is recommended you do not use tags other than <Ignore> more than once.
  • You can use any separator to separate the <Artist> and <Title> tags  but note that in Windows, a filename can't contain any of the following characters \/:*?"<>| . Use "Alt+S" keyboard shortcut to add a stored separator.
  • Also note that the separator "-" (no whitespaces) is different from "- " or " - ". The Filename format has to be an exact match, the program will ingore any file which does not exactly match the Filename Format.

Empty button

Clicking this button will empty any format currently used in the Create/Edit Filename Format box.

Store Format button

Clicking this button will store the format entered in the Create/Edit Filename Format box after checking that it is a valid format. You can also use the Enter key in the Create/Edit Filename Format box to store a format and close the Format Options dialog box.
