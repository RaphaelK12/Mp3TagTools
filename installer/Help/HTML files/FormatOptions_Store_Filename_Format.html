Store Filename Format

You can store many different types of Filename Formats that you have come across in your files. To store a format you can either use the Store Format button or use the Enter key in the Create/Edit Filename Format box to store a format and close the Format Options dialog box.

Delete Format button

Clicking this button will delete the currently selected stored format from the list of stored formats.

Edit Format button

Clicking this button will delete the currently selected stored format from the list of stored formats and place it in the Create/Edit Filename Format box for the user to edit the format. Once you have finished editing the format, store the format as described above.

Move Up button

Clicking this button will move the currently selected format up the order in the stored Filename Format list.

Move Down button

Clicking this button will move the currently selected format down the order in the stored Filename Format list.


Selecting a Filename Format by using the arrow keys or single/double clicking any of the stored Filename Formats will load that format into the Create/Edit Filename Format box.
