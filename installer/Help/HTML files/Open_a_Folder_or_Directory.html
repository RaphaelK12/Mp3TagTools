Open a Folder or Directory

After starting the program, you must select a folder or directory containing the files you want to work with:

Use the shortcut Ctrl+O or Directory button or select Open Directory from the File main program menu.

This will display a Select Working Directory dialog, use the dialog to browse to a folder or directory containing the mp3 files you want to work with. Select any file and just press "Open", the program will list all the files in this folder.

You can check the Recurse subdirectories checkbox so that the program will also search and list all the mp3 files and its tags from all the subdirectories in this folder, any number of levels down.

Note:

For all future tag operations the program will use this specified path and also Create Directories will organize files using this folder.
