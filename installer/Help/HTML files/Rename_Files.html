Rename Files

Before Renaming files select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To rename files directly or using a filename or directory format:

Use the shortcut F2 or Ctrl+R or Rename Files button or select Rename File from the File main program menu.

Direct File Rename

If Use Filename Format checkbox and Use Directory Format checkbox are not checked, using F2 will allow the user to directly edit the filename in the program list.

Custom Format File Rename

To create a Custom Format use the Format Options button.

  •  Use Filename Format: checking this will rename mp3 files based on their tags. For example a mp3 having <Artist> tag "Bee Gees" and <Title> tag "How Deep Is Your Love", when renamed using the filename format "<Artist> - <Title>" will rename the file "Bee Gees - How Deep Is Your Love.mp3". You can use any separators and any combination of formats by using the Format Options box.
  • Use Directory Format: checking this will create folders based on the mp3 tag. For example a  mp3 having <Artist> tag "Bee Gees" and <Album> tag "Greatest Hits" the format "..\Artist\Album" will read the Artist tag and create a folder in the current folder named "Bee Gees", next it will create a folder in the "Bee Gees" folder called "Greatest Hits", the mp3 is then moved into the newly created folder. Directory formats can be used along with Filename Format renaming.
