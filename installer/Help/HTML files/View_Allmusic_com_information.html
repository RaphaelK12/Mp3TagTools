View Allmusic.com information

To view the detailed album information from the excellent allmusic.com website for a mp3, its Artist or Album or Title tag should be complete in ID3v1.1 or ID3v2.x.

The program will use the ID3 tag information to create a search query for the allmusic.com website. If an exact match is not found the site will display a list of possible matches in your browser.

It is recommended to use the Album tag to search allmusic.com, if it has a unique name, (not a generic name like "Greatest Hits") this may open the exact page for the Album on the website. Thus you can read all about the album and fill missing information like Track number, Year of Album release, Genre etc.
