Synchronize Tags

Before Synchronizing tags select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To Synchronize Tags between ID3v1.1 and ID3v2.x:

Use the shortcut F8 or Synchronize Tags from the Tools main program menu.

When Synchronize Tags is used the program will search the ID3v1.1 tags like Artist, Album etc for any empty tags. For example, if the Album tag of ID3v1.1 is empty while the Album tag of ID3v2.x is not empty, then the ID3v2.x Album tag is copied into ID3v1.1. Similarly the empty ID3v2.x tags will be filled using ID3v1.1 information. If both the tags are filled then program will ignore that tag. Thus the missing information in ID3v1.1 and ID3v2.x is filled using each others completed tags.


Note:

When copying tags from ID3v2.x to ID3v1.1 some of the tag information maybe truncated. This is because ID3v1.1 specifications allow only limited amount of information in them. For example Artist tag can contain a maximum of 30 characters, to see the ID3v1.1 specifications visit the ID3 website http://www.id3.org/ .
