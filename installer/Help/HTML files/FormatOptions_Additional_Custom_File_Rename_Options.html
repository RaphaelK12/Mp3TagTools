Additional Custom File Rename Options

Use Leading Zeroes for Track names

Check the Use Leading zeroes checkbox to add a leading 0 before a single and double digit track numbers (ex: 01, 02..) in a Filename. This is useful for the correct display order of filenames with track numbers in Windows Explorer. You can configure the way the leading zeroes are added by changing the values in the Tracks (1-9) and Tracks (10-99) boxes.


Append Bitrate and Append Mode to Filenames

Check the Append Bitrate and Append Mode boxes to add these MPEG information of the file to the end of a Filename.
