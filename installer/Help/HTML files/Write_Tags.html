Write Tags

Before Writing tags select one or more files from the list by clicking in the list while pressing down the Ctrl key or by using the keyboard using Ctrl+Shift+Up arrow or Ctrl+Shift+Down arrow after selecting an item.

To write tags use the shortcut Ctrl+W or Write Tags button or select Write Tags from the Edit main program menu.

Depending on the currently selected Tag info Tab Write Tags will:

 

ID3v1.1 or ID3v2.x Tab

Mass Tagging

If any of the the tag information (Artist, Title etc)  fields in a tab page are enabled, then all the selected mp3s are set with same tags as that entered in the tag information field.

For example you have several mp3s from one Album and want to set the same Album tag to all mp3s. First disable all other fields by unchecking all other checkboxes except the Album checkbox. Enter the tag in the Album field ex: "Dance Album", make sure you have selected the files in the list to be tagged, then use Ctrl+W to write the tags.

Custom Format Tagging

You can use a custom format to write the tags from the mp3 filename or directory names. To create a Custom Format use the Format Options button.

  •  Use Filename Format: checking this will tag mp3 files based on their filenames. For example you have a mp3 named "Bee Gees - How Deep Is Your Love.mp3" by using a filename format like "<Artist> - <Title>" the program will read the filename of that mp3 file and extract all the letters before the " - " separator as the <Artist> tag and all the letters following it as the <Title> tag of the mp3. You can use any separators and any combination of formats by using the Format Options box.
  • Use Directory Format: checking this will tag mp3s based on the name of the folder the file is in. For example the format "..\Artist\Album" will read the name of the folder containing the mp3 and write it to the Album tag, the name of the folder in which this folder is contained is used to write the Artist tag. Directory formats can be used along with mass tagging or filename format tagging.
Note:

When a Filename or Directory format is selected the corresponding tag fields will be disabled by the program. For example using the filename format "<Artist> - <Title>" will disable the Artist and Title tag fields in the tag information page.  This because the information to be written to these tags will be extracted from the filename or folder name, the user should not enter any information for those fields.

Lyrics Tab

Only the lyrics of the mp3 will be written.

Picture Tab

Each mp3 can have more than one picture, of a particular specified Picture Type and Picture Description.

The default Picture Type used is "Other" you can change it another like "Cover ( front)" using the Picture Type combo box.

Next use Load Picture Button, and browse to the location of the picture file (ID3 group recommends JPEG or PNG files to keep tag sizes to a minimum), click Open and a preview of the file will load.

Enter a Picture Description or leave it empty. Now use Write Tags button to add the currently displayed picture to the mp3. Change Picture Type and repeat the steps to add another picture.
